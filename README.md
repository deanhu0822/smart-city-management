# NYC Smart City Nexus

**Energy, Waste & Urban Intelligence Platform**

> Spark Hack NYC 2026 | Environmental Impact Track | NVIDIA GB10

A decision-support platform that merges NYC's energy and waste data into one unified intelligence layer. Built to run entirely on-device using NVIDIA RAPIDS, NIMs, and cuOpt on the Acer Veriton GN100.

## Tech Stack

| Layer | Tool | Purpose |
|-------|------|---------|
| Data Processing | RAPIDS cuDF + cuSpatial | GPU-accelerated data processing (6M+ rows) |
| Route Optimization | RAPIDS cuOpt | Waste collection vehicle routing on GPU |
| LLM Inference | NVIDIA NIMs (Llama 3.1 8B) | Local site scoring and natural language insights |
| Simulation | Custom Python (GPU-vectorized) | BESS 24h charge/discharge dispatch |
| Dashboard | Streamlit + Folium | Interactive map, charts, scenario planner |
| Hardware | Acer Veriton GN100 / NVIDIA GB10 | 128 GB unified memory, 1 PFLOP FP4 |
